# git_test

## init
```
$ git init
```

## add
* only file
```
$ git add <file>
```
* all file
```
$ git add .
```

## commit
```
$ git commit -m "first commit"
```

## branch
```
$ git branch -M main
```

## remote
```
$ git remote add origin git@github.com:zangmen/git_test.git
```

## push
```
$ git push -u origin main
```
